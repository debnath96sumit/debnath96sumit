## 🎯 Featured Projects

### Multi-Provider AI Chat App | [Backend](https://github.com/debnath96sumit/AI-chat-backend) | [Frontend](https://github.com/debnath96sumit/AI-chat-frontend)

A full-stack AI chat application with a Strategy-pattern LLM provider layer supporting Groq, Gemini, and Hugging Face. Features streaming responses, PDF/TXT file uploads with text extraction, GitHub OAuth, and a multi-model switching UI.

**Why built:** I wanted to understand how to build a clean provider abstraction over LLMs — one that doesn't leak provider-specific quirks into your business logic.

**Interesting decisions:**
- Strategy pattern for provider-agnostic streaming
- Passport-based GitHub OAuth with session handling
- File-to-context pipeline for document-grounded conversations

**Tech Stack:** NestJS, TypeScript, MongoDB, Passport, Groq SDK, Google Generative AI
